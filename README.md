## Deskripsi Proyek
**EABOT** adalah repositori berbasis *Expert Advisor* atau Bot Trading yang ditulis sepenuhnya dalam [MQL5](https://www.mql5.com/), bahasa pemrograman spesifik untuk pengembangan strategi perdagangan algoritmik pada platform MetaTrader 5. 

Proyek ini bertujuan untuk membantu pengguna mengotomatisasi perdagangan di pasar keuangan dengan menggunakan strategi perdagangan yang dapat disesuaikan sepenuhnya.

## Fitur Utama
- **Otomatisasi Perdagangan**: Mengotomatisasi proses pembukaan, penutupan, dan pengelolaan perdagangan.
- **Konfigurasi Fleksibel**: Mendukung berbagai pengaturan strategi perdagangan seperti manajemen risiko, parameter indikator, dan lainnya.
- **Kompatibilitas MetaTrader 5**: Dikembangkan sepenuhnya untuk digunakan dengan platform perdagangan MetaTrader 5.

## Prasyarat
Sebelum memulai, pastikan Anda memiliki:
1. Akun trading yang mendukung MetaTrader 5.
2. Platform trading MetaTrader 5 terinstal pada PC.
3. Pengetahuan dasar tentang pengoperasian EA (Expert Advisor) pada MetaTrader.

## Instalasi
Ikuti langkah-langkah berikut untuk menginstal EABOT pada MetaTrader 5:
1. Unduh file MQL5 dari repositori ini.
2. Buka platform MetaTrader 5.
3. Klik menu `File` > `Open Data Folder`.
4. Masuk ke folder `MQL5/Experts`.
5. Salin file `.mq5` dari repositori ini ke folder tersebut.
6. Restart platform MetaTrader 5.
7. Tambahkan EABOT ke grafik trading Anda melalui tab Navigator.
